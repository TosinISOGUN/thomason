# Thomason — Coffee, Quietly Perfected

Thomason is a premium, minimalist web experience designed for a specialty coffee shop. It emphasizes a "quiet ritual" brand philosophy through a refined aesthetic, high-quality imagery, and smooth, intentional animations.

## ✨ Features

- **Atmospheric Hero Section**: Smooth-fading slideshow featuring professional coffee and interior photography.
- **Brand Story**: A narrative-driven section detailing the shop's philosophy with large-scale imagery.
- **Curated Menu**: Interactive signature drink showcase with hover effects and price listings.
- **Space Gallery**: A grid-based look into the shop's interior design and atmosphere.
- **Follow Along**: Integrated Instagram feed styling to showcase the shop's digital presence.
- **Newsletter Subscription**: A minimalist signup form to keep customers connected.
- **Responsive Branding**: Dynamic logo placement that adapts to mobile and desktop layouts, including a sticky navigation bar that reveals the logo upon scrolling.
- **SEO Optimized**: Includes semantic HTML, meta tags for social sharing, and JSON-LD structured data for local business visibility.

## 🛠️ Technology Stack

- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [Shadcn UI](https://ui.shadcn.com/) (Radix UI)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Type Safety**: [TypeScript](https://www.typescriptlang.org/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components`: UI components organized by section (Hero, Menu, Visit, etc.).
- `src/pages`: Main application views.
- `src/assets`: Image and branding assets.
- `src/hooks`: Custom React hooks for shared logic.
- `public`: Static assets including the business logo and favicon.

## 🎨 Design System

The project uses a custom color palette defined in `tailwind.config.ts`:
- **Primary**: Rich espresso tones.
- **Secondary**: Warm, creamy beige.
- **Accent**: Deep roasted browns.
- **Typography**: A mix of Playfair Display (Serif) for headings and DM Sans (Sans) for body text to balance elegance and readability.
